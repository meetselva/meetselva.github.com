attrchange is a simple jQuery plugin to bind a listener function when an element's attribute is added, removed or modified. The event object passed to the handler function will have the attribute change details.

How it works?

The plugin chooses one of the following best browser supported methods to detect an attribute change. On attribute change, the plugin triggers the handler function with attribute change details.

  • Mutation Observer - New Web Standard to react to changes in DOM
  • DOMAttrModified - Now deprecated, but included to support older browser
  • onpropertychange - Deprecated IE specific method (since IE9), but included to support IE < 9

Usage

$([selector]).attrchange({
  trackValues: true, 
  callback: function (event) {
	//event.attributeName - Attribute Name
	//event.oldValue - Prev Value
	//event.newValue - New Value
  }
});
