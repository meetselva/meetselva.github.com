attrchange is a simple jQuery plugin to bind a listener function on attribute change of an element. The handler function is triggered when an attribute is added, removed or modified to the element.

This plugin utilizes different DOM events API's (Mutation Observer / DOMAttrModified / onpropertychange) to detect attribute changes. It does not use "Polling", which is slower and may cause performance issue if not used with care.

The main goal of this plugin is to provide cross browser compliant solution to handle DOM changes. The plugin bundles with three different techniques and chooses the best supported technique to effectively trigger the handler.

Features

  • Cross browser support
  • Compact and light weight
  • Utilizies DOM Events API to detect Attribute Changes
  • New Disconnect feature allows to disconnect the handler
  • Support for Extensions

Usage

$([selector]).attrchange({
  trackValues: true, 
  callback: function (event) {
	//event.attributeName - Attribute Name 
	//event.oldValue - Prev Value
	//event.newValue - New Value
  }
});
